# Simple Interest Calculator
A shell script to calculate simple interest given principal, annual rate of interest and time period in years.

## Input:
* p, principal amount
* t, time period in years
* r, annual rate of interest

## Output:
* simple interest = p*t*r
A simple tool to calculate the interest earned or paid on a principal amount over a specific period.

## Description
This project provides a clear understanding of how simple interest is calculated. It can be used as a basic script or a web-based tool for financial planning.

## Formula
The interest is calculated using the standard formula:

$$A = P(1 + rt)$$

Where:
* **A**: Total amount (Principal + Interest)
* **P**: Principal amount
* **r**: Annual interest rate (decimal)
* **t**: Time period (in years)

## How to Use
1. Input the **Principal** amount.
2. Input the **Annual Interest Rate**.
3. Input the **Time** in years.
4. The calculator will output the total interest and the final balance.
